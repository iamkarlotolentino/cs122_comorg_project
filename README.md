# Personal Bank Account Manager

#### THIS PROJECT

An automated teller machine mobile designed to imitate an industrial bank machine. This is a project requirement in our CS-122 course subject at Mariano Marcos State University. To demonstrate how we can create a menu-driven interface applications using vanilla assembly language (ASM). 

#### Video Pages
The application changes video page for faster view response. Refer in this list to identify which page is being displayed and has been dedicated to.
```
[0] --> Login page
[1] --> Menu page
[2] --> Input amount page
[3] --> Reset PIN page
[4] --> Balance page
[5] --> Details page
[6] --> Open page (for any purposes)
[7] --> Please wait page
```
#### Account File Format
There are three important data in the text file: (1) ATM Card No. (2) Pin code (3) Balance.
An example of the text file content below:
```
0123456789012345
123456
000000000000
```
Each data is separated by `new line`.
- 16 digits for ATM Card No.
- 6 digits for Pin code
- 12 digits for Balance
